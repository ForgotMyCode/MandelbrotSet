# Mandelbrot Set

Just a fun project. Interactive real-time 2D visualization of the Mandelbrot set in C++.

![Screenshot](/screenshot.png?raw=true "Screenshot")

## Requirements

 - C++20 with Cmake
 - OpenGL
 - GLFW
 - OpenMP
 - Threads

## Notes

Edit the CMakeFiles.txt!

The GUI was made using Dear Imgui.
